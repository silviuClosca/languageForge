## Getting Started

### Installation

1. Install the add-on from AnkiWeb or manually place it in your Anki addons folder
2. Restart Anki
3. Access LanguageForge via **Tools ‚Üí LanguageForge ‚Äì Language System**

### First Launch

On first launch, LanguageForge automatically:
- Creates a `user_data` folder to store your data
- Initializes a "Default" profile
- Sets up the dashboard with empty goals, tracker, and resources

### Basic Navigation

LanguageForge has **5 main tabs**:
1. **Dashboard** - Overview of all features
2. **Tracker** - Detailed daily/weekly/monthly activity tracking
3. **Goals** - Full goal management with subtasks
4. **Resources** - Learning materials library
5. **Settings** - Themes, profiles, and preferences

## Monthly Goals

**Purpose:** Set and track up to 3 major goals per month with subtasks.

### Goal Structure

Each goal includes:
- **Title** (e.g., "Complete B1 Course")
- **Category** (General, Vocabulary, Grammar, etc.)
- **Subtasks** (break goal into actionable steps)
- **Reflection** (notes, insights, challenges)
- **Timestamps** (created/completed dates)

### How to Set Goals

1. **Select Month:**
   - Use dropdown to choose current or future month
   - Each month has 3 goal slots

2. **Write Goal Title:**
   - Click title field, type your goal
   - Auto-saves on changes

3. **Choose Category:**
   - Select from dropdown (General, Vocabulary, Grammar, Reading, Listening, Speaking, Writing)

4. **Add Subtasks:**
   - Click "Add Subtask" button
   - Write specific, actionable steps
   - Check boxes as you complete them
   - Delete with ‚ùå button

5. **Complete Goal:**
   - Check the main checkbox when fully done
   - Completion timestamp recorded automatically

### Best Practices

- **SMART goals:** Specific, Measurable, Achievable, Relevant, Time-bound
- **Break down big goals:** Use 4-8 subtasks per goal
- **Review monthly:** Set aside time to plan next month's goals
- **Use reflections:** Document what worked, what didn't, lessons learned

### Archive Feature

Past months are automatically archived and marked as read-only. You can still view them for reference.

## Learning Resources

**Purpose:** Centralized library for all your learning materials.

### Resource Fields

- **Title** (required)

   - Changes auto-save

2. **Open URLs:**
   - Click "Open" button to launch in browser
   - Works with any valid HTTP/HTTPS link

3. **Search/Filter:**
   - Use search box to filter by title or notes
   - Real-time filtering

4. **Delete:**
   - Click ‚ùå button to remove a resource
   - Confirmation required

### Organization Tips

- **Tag in notes:** Add #tags for easier searching (e.g., "#grammar #beginner")
- **Status tracking:** Keep Status updated as you progress
- **URL everything:** Add YouTube playlists, course links, PDFs hosted online
- **Regular cleanup:** Archive completed resources or mark as Reference

## Fluency Snapshot (Radar Chart)

**Purpose:** Visualize and track overall language proficiency across four skills.

### Understanding the Chart

- **4 axes:** Reading, Listening, Speaking, Writing
- **Scale:** 1 (beginner) to 5 (advanced)
- **Ghost polygon:** Previous month's values (if available)
- **Trend arrows:** ‚Üë improved, ‚Üì declined, = same
- **Balance Index:** 0-100% (higher = more balanced skills)

### How to Update

1. **Select Month:**
   - Use dropdown (current month or past)

2. **Adjust Values:**
   - Click and drag any skill axis
   - Or click axis labels to set directly
   - Values snap to 1-5 scale

3. **Save:**
   - Click "Save Snapshot" button
   - Timestamps recorded automatically

### Self-Assessment Guide

| Level | Reading | Listening | Speaking | Writing |
|-------|---------|-----------|----------|---------|
| **1 - Beginner** | Recognize basic words | Catch a few familiar words | Struggle with basic phrases | Write simple words/phrases |
| **2 - Elementary** | Understand simple texts | Follow slow, clear speech | Express basic needs | Write short messages |
| **3 - Intermediate** | Read short articles | Understand main points in conversations | Discuss familiar topics | Write coherent paragraphs |
| **4 - Upper-Intermediate** | Comprehend complex texts | Follow fast native speech | Express ideas fluently | Write detailed texts |
| **5 - Advanced** | Read any material comfortably | Understand subtle nuances | Speak like a native | Write professional-level content |

### Tips

- **Monthly snapshots:** Update at month-end for consistency
- **Be honest:** Self-assessment accuracy improves tracking
- **Focus on balance:** A score of 3/3/3/3 beats 5/5/1/1
- **Compare trends:** Look for patterns over 3-6 months

## Multi-Profile System

**Purpose:** Manage separate data for multiple languages.

### Profile Features

- **Isolated data:** Each profile has its own goals, tracker, resources, radar, daily plan
- **Global settings:** Theme and font size are shared across profiles
- **Easy switching:** Dropdown in top-right corner
- **Persistent:** Remembers last active profile

### How to Use Profiles

#### Create a Profile

1. Go to **Settings ‚Üí Profile Management**
2. Click **‚ûï Add Profile**
3. Enter a name (e.g., "Spanish", "Japanese üáØüáµ", "French")
4. Profile appears in list and dropdown

#### Switch Profiles

1. Use dropdown selector (top-right corner)
2. All tabs reload with that profile's data
3. Active profile shown with ‚≠ê in Settings

#### Rename a Profile

1. Settings ‚Üí Profile Management
2. Select profile from list
3. Click **‚úèÔ∏è Rename**
4. Enter new display name
   - Display name changes, folder name stays the same (safe)

#### Delete a Profile

1. Settings ‚Üí Profile Management
2. **Switch to a different profile first** (can't delete active profile)
3. Select profile to delete
4. Click **üóëÔ∏è Delete**
5. Confirm deletion
   - **Warning:** This permanently deletes all data for that profile

### Profile Limits

- **Maximum profiles:** 50
- **Name length:** 1-30 characters
- **Cannot delete:** Default profile or currently active profile

### Use Cases

- **Multiple languages:** Spanish profile, Japanese profile, etc.
- **Different learning stages:** "Spanish A2" vs "Spanish B2"
- **Separate courses:** "University Spanish" vs "Self-Study Spanish"
- **Testing:** Create a "Test" profile for experimenting

## Tips & Best Practices

### Daily Routine

1. **Morning:** Open Dashboard, review daily plan tasks
2. **During study:** Mark tracker circles as you practice each skill
3. **Evening:** Update daily plan for tomorrow, reflect on progress

### Weekly Routine

1. **Sunday/Monday:** Review past week's consistency
2. Set daily plan tasks for the week
3. Adjust goals if needed

### Monthly Routine

1. **Month-end:**
   - Complete monthly goals or carry over
   - Write reflections on each goal
   - Update radar snapshot
   - Archive past month

2. **Month-start:**
   - Set 3 new goals with subtasks
   - Plan high-priority resources
   - Review last month's radar for weak skills

### Maximizing the Add-on

- **Be consistent:** Daily tracking is more valuable than perfect tracking
- **Start small:** 1-2 goals with 3-4 subtasks each
- **Review regularly:** Weekly check-ins prevent drift
- **Use all features:** Tracker + Goals + Resources + Radar work together
- **Iterate:** Adjust your system as you learn what works

## Data Storage & Backup

### File Structure

```
languageforge/
  user_data/
    profiles.json          # Profile registry + active profile
    settings.json          # Global settings
    profiles/
      default/
        goals_v2.json      # Monthly goals data
        tracker.json       # Daily activity data
        resources.json     # Learning resources
        radar.json         # Skill snapshots
        dailyplan.json     # Daily plan tasks
      spanish/
        [same files]
      japanese/
        [same files]
```

### Backup Recommendations

**Option 1: Manual Backup**
1. Close Anki
2. Copy entire `user_data` folder
3. Store in cloud storage or external drive

**Option 2: Git/Version Control**
- Add `user_data/` to your Anki backups
- Commit regularly to track changes

**Frequency:**
- **Weekly:** If actively using
- **Monthly:** At minimum

### Restoring Data

1. Close Anki
2. Replace `user_data` folder with backup
3. Restart Anki

## Troubleshooting

### LanguageForge won't open

**Solution:**
1. Check for error messages in Anki console (Tools ‚Üí Add-ons ‚Üí View Files)
2. Restart Anki
3. Disable other add-ons to check for conflicts

### Data not saving

**Symptoms:** Changes disappear after switching tabs or restarting

**Solution:**
1. Check file permissions on `user_data/` folder
2. Ensure Anki has write access
3. Check if files are read-only

### Profile switching not working

**Symptoms:** Same data shows in all profiles

**Solution:**
1. Verify profile folders exist in `user_data/profiles/`
2. Check `profiles.json` for active profile ID
3. Restart Anki to refresh profile system

### Radar chart not updating

**Solution:**
1. Click "Save Snapshot" button after changing values
2. Switch to a different tab and back
3. Check `radar.json` file exists in current profile

### Theme not applying

**Solution:**
1. Settings ‚Üí Visual Settings ‚Üí select theme again
2. Try "Anki Auto" mode
3. Restart Anki

### Search not working in Resources

**Solution:**
1. Clear search box and try again
2. Check for special characters
3. Use simpler search terms
